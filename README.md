## 安装教程

1. 准备：[TRSS-Yunzai](../../../Yunzai) 和 [Philia](../../../Philia)
2. 输入：`#安装Philia-Plugin`
3. 退出，终端输入：`pnpm link [Philia安装路径]`
4. 启动，输入：`#Philia设置`
